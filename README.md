## 🛠 Features
- **Modern Design**: Premium dark blue aesthetic with glassmorphic cards and responsive layout.
- **MathJax Integration**: High-quality rendering of mathematical equations and algebraic structures.
- **Dynamic Timeline**: Professional history and education mapping.
- **Social Integration**: Direct links to Google Scholar, ORCID, ResearchGate, and LinkedIn.

## 📁 Project Structure
- `index.html`: Main website structure and content.
- `style.css`: Custom design tokens, animations, and glassmorphism effects.
- `script.js`: Interactive elements and scroll animations.
- `profile.png`: Profile avatar.

## 📝 How to Update
1.  **Content**: Edit the text directly in `index.html`.
2.  **Equations**: Use standard LaTeX syntax within `$ ... $` delimiters for MathJax to render.
3.  **Styles**: Modify colors or fonts in `style.css`.
4.  **Photo**: Replace `profile.png` with your preferred headshot.
